Curriculum

2일 · 14시간 · 자동화 마스터 여정

Day 1은 "코딩 없이도 가능"을 증명하고, Day 2는 본격 멀티에이전트 시스템 개발로 도약합니다.

Day 1 · 기초 + DMAP 7H

"코딩 없이도 만들 수 있다"를 증명하는 날

  • 엔터프라이즈 AI 서비스 이해 1H
    AI 개요·3대 필요성·5대 핵심 구성요소·AI 아키텍처. AI 도구 5-Layer 지도(Foundation Model → Cloud AI PaaS)
  • AI 도구 기본 사용법 2H
    Claude Code(AGENTS.md, MCP) · Codex · 프롬프트 5대 구성. 스킬·에이전트·플러그인 이해
    스킬·에이전트 실습 MCP 추가
  • 업무 자동화 플러그인 개발 (DMAP) 4H
    DMAP 플러그인 설치·사용. 코딩 없이 스킬·에이전트·플러그인 개발. 웹+유튜브 검색 레포트 플러그인 + 개인 업무용 플러그인 제작
    레포트 플러그인 개인 업무 플러그인
Day 2 · MAS + Abra 7H

"멀티에이전트로 도약"하는 날

  • MAS 이해 (LangGraph) 3H
    MAS 정의·필요성·구축방안(SAS패턴, 외부 툴 통신, 상태 관리). LangGraph + 하네스 엔지니어링(비용·성능·보안). 실습: 질문 분류 → Code 생성(벡터DB 검색→재시도) / Q&A(내부DB+웹+YouTube) → Fallback 처리
    분류·라우팅 에이전트 Fallback 처리
  • 업무자동화 MAS 개발 (Abra) 4H
    Abra 플러그인으로 기획(유저스토리)→설계(Dify DSL 자동생성)→개발(RAG+LangGraph MAS+프론트엔드)→배포(Docker Compose)까지. 실습 서비스: 업종/지역/시즌 트렌드 분석 → 예산별 이벤트 생성 → 우리동네TV·링고비즈·지니TV 다중 채널 배포 → 성과 분석
    Dify DSL → 프로토타입 LangGraph MAS Docker 배포

Built-in AI Toolkit · 2 Plugins

본 과정의 핵심 도구 — DMAP & Abra

강사의 경험과 노우하우가 집약된 두 개의 플러그인을 실습 전반에 활용합니다. 단계별 학습 곡선에 맞게 설계되어, 초보자도 첫날부터 결과물을 만들 수 있습니다.

Day 1 · Non-Coding Automation

DMAP 플러그인

코딩 없는 플러그인 개발 가속기

코딩 없이 스킬·에이전트·플러그인을 만들 수 있도록 지원하는 강사의 경험과 노우하우가 집약된 플러그인. AI를 처음 다루는 직원도 첫날에 본인 업무 자동화 도구를 직접 만들 수 있습니다. 첫 실습 산출물은 "웹+유튜브 검색하여 레포트 작성하는 플러그인".

Day 2 · Multi-Agent System

Abra 플러그인

Dify × LangGraph MAS 빌더
